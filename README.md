## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Clear the app

   ```bash
   npm run reset-project
   ```

3. Start the app

   ```bash
   npx expo start
   ```

# Toast Message 
```bash
npm install react-native-toast-message
```
# supabase
```bash
npm install @supabase/supabase-js
```
# storage
```bash
 npx expo install @react-native-async-storage/async-storage
```
